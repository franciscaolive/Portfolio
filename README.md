# Francisca Miranda - Portfolio Website

A responsive bilingual portfolio website showcasing selected design and game projects.

## Live Website

[Open Portfolio](https://franciscaolives.github.io/Portfolio/)

## Overview

This repository contains a static front-end portfolio built with HTML, CSS, and vanilla JavaScript. It includes:

- Responsive layout for desktop and mobile
- Language switching (Portuguese and English)
- Light and dark theme toggle
- Project filtering by category
- Individual project detail pages
- Language-aware CV download links

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- JSON-based content and localization
