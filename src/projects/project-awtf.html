Away with the Fairies is a poster developed in Adobe Illustrator from an observational starting point: a gate on the FEUP campus whose ornamental structure suggested a strong graphic language. Using the Rectangle Grid Tool as a generative base, the project translates that architectural reference into a more stylised visual composition.

The work explores how a real element from the built environment can be abstracted into a creative graphic system, balancing repetition, symmetry, and decorative rhythm with a more contemporary poster sensibility. Rather than reproducing the gate literally, the intention was to reinterpret it and build an image that feels both structured and atmospheric.

As an exercise, the project reflects an interest in pushing visual references beyond documentation and towards authorship, treating design as a space for interpretation, refinement, and aesthetic ambition.

It is a small but formative study in observation, abstraction, and composition, and in using digital tools not only for execution, but for discovering visual possibilities within an initial idea.

Away with the Fairies é um cartaz desenvolvido em Adobe Illustrator a partir de um ponto de observação concreto: um portão no campus da FEUP cuja estrutura ornamental sugeriu uma linguagem gráfica forte. Recorri à Rectangle Grid Tool como base geradora, traduzindo essa referência arquitetónica para uma composição visual mais estilizada.

O trabalho explora a forma como um elemento real do espaço construído pode ser abstraído para um sistema gráfico criativo, equilibrando repetição, simetria e ritmo decorativo com uma sensibilidade de cartaz mais contemporânea. Em vez de reproduzir literalmente o portão, a intenção foi reinterpretá-lo e construir uma imagem simultaneamente estruturada e atmosférica.

Enquanto exercício, o projeto reflete um interesse em levar as referências visuais para além da documentação e em direção à autoria, tratando o design como espaço de interpretação, depuração e ambição estética.

É um estudo pequeno mas formativo sobre observação, abstração e composição, e sobre o uso de ferramentas digitais não apenas para executar uma ideia, mas também para descobrir possibilidades visuais dentro dela.
